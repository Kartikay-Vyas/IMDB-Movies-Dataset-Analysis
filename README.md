# 🎬 IMDB Movies Dataset Analysis

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on an IMDB Movies dataset using Python. The objective of the analysis is to uncover trends related to movie genres, ratings, financial performance, and yearly release patterns.

The project includes data cleaning, visualization, univariate analysis, bivariate analysis, multivariate analysis, and insight generation to better understand factors influencing movie success.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

---

## 📂 Dataset Features
The dataset contains movie-related information such as:
- Movie Name
- Genre
- Release Date
- IMDB Score
- Budget
- Revenue
- Original Language
- Country
- Crew Information

---

## 🔍 Analysis Performed

### ✅ Data Cleaning
- Handled missing values
- Converted date columns into datetime format
- Extracted year from release date
- Checked duplicates and outliers

### ✅ Univariate Analysis
- Distribution of movie scores
- Genre frequency analysis
- Trend visualization using histograms and bar charts

### ✅ Bivariate Analysis
- Budget vs Rating analysis
- Genre-wise rating comparison
- Correlation analysis between financial variables

### ✅ Genre-Specific Analysis
- Highest-rated genres
- Genre popularity over time
- Rating distribution across genres

### ✅ Year & Trend Analysis
- Average ratings over years
- Number of movie releases per year
- Industry growth trend analysis

### ✅ Multivariate Analysis
- Decade-wise genre popularity
- Correlation heatmap
- Genre and year impact on ratings

---

## 📊 Key Insights
- Movie production has increased significantly over the years.
- Certain genres consistently achieve higher average ratings.
- Budget and revenue show a strong positive correlation.
- Higher budgets do not always guarantee higher audience ratings.
- Audience preferences change noticeably across decades.
